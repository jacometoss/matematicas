Aplicando la fórmula de la ecuación de segundo grado

Ejemplo 1

Resolver la diferencia de cuadrados siguiente:

x^{2}-16 = 0

Los coeficientes son: 

  • a = 1
  • b = 0
  • c = -16

Sustituyendo los coeficientes en la fórmula de segundo grado se obtiene lo siguiente:

x = \frac{-b\pm\sqrt{b^{2}-4ac}}{2a}

x = \frac{-\left ( 0 \right )\pm\sqrt{\left ( 0 \right )^{2}-4\left ( 1 \right )\left ( -16 \right )}}{2\left ( 1 \right )}

x = \frac{-\left ( 0 \right )\pm\sqrt{64}}{2}

Finalmente, las raíces de la ecuación cuadrática son:

x = \frac{-\left ( 0 \right )\pm 8}{2} = \pm 4
